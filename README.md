## 📌 Sobre o projeto

Implementação do clássico **jogo da memória**: encontre todos os 8 pares no menor número de jogadas e tempo. As cartas embaralham a cada partida, viram com animação 3D (CSS `transform`) e a vitória abre um modal com seu desempenho.

## ✨ Funcionalidades

- 🃏 16 cartas (8 pares) **embaralhadas** a cada partida
- 🔄 Animação de virada em **3D** (flip com `rotateY`)
- 🎯 Contador de **jogadas**, **cronômetro** e **pares encontrados**
- 🏆 Tela de vitória com resumo do desempenho
- 📱 Responsivo e jogável no celular

## 🛠️ Tecnologias

HTML5 · CSS3 (3D transforms, grid) · JavaScript (lógica de jogo, sem bibliotecas)
